Publication data analysis using Hadoop-MapReduce

The project aims at analyzing DBLP publication datasets to gain insights about the relationship between authors, publications, venues etc. It uses Apache Hadoop's Map Reduce framework for parallel of the processing the data and AWS EMR for deployment on cloud.

Cloud datacenter simulation

This project simulates the working of datacenter and various service implementation in cloud computing. The project uses Cloud Sim plus framework to simulate the model and is developed using Scala. It shows various implementation of scheduling and allocation policies and their effects on execution time, provisioning of resources.It also shows the result of configuring parameters and their effect on the allocation of VM and cloudlets.

Web Search Engine

A search engine to retrieve web pages under UIC domain.Used Scrapy to create a multi-threaded crawler scrape ~3500 web pages Implemented a vector space model to compute the similarity between the user query and web pages and integrated it with Page Rank algorithm to generate the most relevant web pages

Chicago Crime Prediction

Chicago has one of the highest crime rate.This projects explores the crime pattern in Chicago and predicts two things.
1)Possibility of arrests - Implemented a logistic regression model with accuracy of 89.29% and F1 - score of 88.44%.
2)Type of crime - Claasified the crime groups using Naives Bayes and achieved an accuracy and F1 score of 99%.

Driverless Vehicles Impact on Taxi Services

The project simulates the working of a ride hailing algorithm to study the impact of crowd-sourced and autonomous vehicles. Implemented Stable marriage and Hungarian Algorithm to assign requests and match the cab with the customer.
Observed an increase in benefit per taxi from crowd-sourced vehicles to autonomous vehicles by 52%.
